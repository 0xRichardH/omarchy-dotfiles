# Plan: Update Ghostty Keybindings to Use ALT Modifier

## Overview
Replace `super` modifier with `alt` in Ghostty keybindings to avoid conflicts with Hyprland window manager shortcuts.

## Current State
- Ghostty config at: `ghostty/.config/ghostty/config`
- User has keybindings using `super` that conflict with system-wide Hyprland shortcuts
- ALT modifier is confirmed unused in Hyprland bindings

## Implementation

### Step 1: Update Ghostty Config
Edit `ghostty/.config/ghostty/config` to add keybindings section with ALT modifier:

Replace `super` with `alt` in all keybindings:
- Window switching (1-9): `alt+physical:one` through `alt+physical:nine`
- Tmux commands: `alt+c`, `alt+g`, `alt+j`, `alt+l`, `alt+u`, `alt+t`, `alt+w`
- Neovim commands: `alt+p` (GotoFile), `alt+f` (Grep), `alt+q` (quit all)

The keybindings will be:
```
# switch to window by index (1-based)
keybind = alt+physical:one=text:\x131
keybind = alt+physical:two=text:\x132
keybind = alt+physical:three=text:\x133
keybind = alt+physical:four=text:\x134
keybind = alt+physical:five=text:\x135
keybind = alt+physical:six=text:\x136
keybind = alt+physical:seven=text:\x137
keybind = alt+physical:eight=text:\x138
keybind = alt+physical:nine=text:\x139

# a-z
# Change to tmux copy mode
keybind = alt+c=text:\x13[
# <prefix>+g (open lazygit window)
keybind = alt+g=text:\x13\x67
# <prefix>+j (open the session manager popup)
keybind = alt+j=text:\x13\x54
# <prefix>+l (switch to last session)
keybind = alt+l=text:\x13\x4c
# <prefix>+u (open url)
keybind = alt+u=text:\x13\x75
# :GotoFile
keybind = alt+p=text:\x3a\x47\x6f\x54\x6f\x46\x69\x6c\x65\x0a
# :Grep (find)
keybind = alt+f=text:\x3a\x47\x72\x65\x70\x0a
# :qa!
keybind = alt+q=text:\x3a\x1b\x1b\x3a\x71\x61\x21\x0a
# <prefix>+c (new window)
keybind = alt+t=text:\x13\x63
# <prefix>+x (close window)
keybind = alt+w=text:\x13\x78
```
